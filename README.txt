Petunjuk Menjalankan Project MiniZinc IDE:
1) Buka aplikasi MiniZinc IDE terlebih dahulu
2) Pada menu bar File, pilih Open, lalu pilih file minizinc.mzn
3) Untuk menjalankan code, klik Run (Memberikan 1 solusi saja)
4) Untuk menampilkan lebih dari 1 solusi, maka klik 'Show configuration editor' pada menu di atas kanan
5) Pada Solving, pilih 'User defined behavior'
6) Pada 'Satisfication problems', terdapat input bar yang dapat diganti value nya menjadi jumlah solusi yang ingin ditampilkan
7) Untuk menampilkan semua solusi, maka dapat uncheck pada tulisan 'Stop after this many solutions (uncheck for all)'
8) Selanjutnya dapat mengklik Run lagi untuk menampilkan solusi

Petunjuk Menjalankan Project Python:
1) Sebelum menggunakan python, pastikan folder MiniZinc berada pada path 'Environment Variables'
2) Untuk menambahkan path, dapat dilakukan pada menu System Properties > Advanced > Environment Variables pada Windows
3) Lalu klik edit pada PATH dan tambahkan directory menuju folder MiniZinc
4) Tahap berikutnya adalah buka python dan Open file python.py
5) Setelah file terbuka, maka tinggal jalankan code nya untuk menampilkan solusi
